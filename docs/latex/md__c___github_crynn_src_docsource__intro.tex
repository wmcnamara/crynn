\mbox{\hyperlink{namespace_crynn}{Crynn}} is an open source game engine created with C++17 and Open\+GL. It is very barebones as it stands. \mbox{\hyperlink{namespace_crynn}{Crynn}} is not overtly user friendly. Knowledge of matrix math, event systems, Open\+GL and G\+L\+FW is required.\hypertarget{md__c___github_crynn_src_docsource__intro_autotoc_md1}{}\doxysubsection{Dependencies}\label{md__c___github_crynn_src_docsource__intro_autotoc_md1}
\mbox{\hyperlink{namespace_crynn}{Crynn}} relies on a multitude of dependencies. They are listed here\+:\hypertarget{md__c___github_crynn_src_docsource__intro_autotoc_md2}{}\doxysubsubsection{Crynn Depends On\+:}\label{md__c___github_crynn_src_docsource__intro_autotoc_md2}

\begin{DoxyItemize}
\item \href{https://www.glfw.org/}{\texttt{ G\+L\+FW}} for windowing, and input.
\item \href{https://github.com/Dav1dde/glad}{\texttt{ G\+L\+AD}} for loading Open\+GL functions.
\item \href{https://github.com/ocornut/imguiDear}{\texttt{ Dear I\+M\+G\+UI}} for G\+UI.
\item \href{https://glm.g-truc.net/0.9.8/index.html}{\texttt{ G\+LM}} for matrix and other graphics related mathematics.
\item \href{https://github.com/nothings/stb}{\texttt{ S\+T\+B\+\_\+\+I\+M\+A\+GE}} for image loading.
