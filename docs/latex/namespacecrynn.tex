\doxysubsection{Detailed Description}
Main Engine Namespace 

This class contains object oriented implementations of various buffers used in Open\+GL.

The default constructor for \mbox{\hyperlink{classcrynn_1_1_v_a_o}{VAO}} allocates memory. They all call the corresponding gl\+Gen function for the buffer they represent, when you construct them.
