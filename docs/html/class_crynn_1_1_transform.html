Detailed Description

Matrix representation with easy functions to transform it. This class is mostly intended to be inherited from. Any transformations made, (translate, scale rotate) will be applied to the "transformMatrix" member. You can use this matrix however you please. Its generally used as a model or view matrix, and works very well with MeshRenderer.
